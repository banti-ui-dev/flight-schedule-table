Flight Schedule Management Table

Features
Virtual scrolling using react-window
Search (flight number, origin, destination)
Filters (status, AOC, body type, days)
Inline editing (STD, STA, status)
Single & bulk delete
Responsive UI
🛠 Tech Stack
React (Vite + TypeScript)
react-window

Setup
npm install
npm run dev

Notes
All data handled client-side
No backend required
